# Instagram Mini Clone

A basic full-stack web application demonstrating user signup, login, and post creation.

## Tech Stack
- Node.js
- Express.js
- HTML
- JavaScript

## Features
- User Signup
- User Login
- Create Text Posts
- View All Posts
- Frontend served using Express static files

## Project Structure
- `server.js` – Backend server and APIs
- `public/` – Frontend files (HTML & JS)
- `users.json` – Stores user data
- `posts.json` – Stores post data

## How to Run the Project
1. Install dependencies  
    npm install

2. Start the server  
    node server.js

3. Open in browser  
    http://localhost:3000

## Notes
- JSON files are used for simplicity instead of a database.
- The project focuses on core functionality and frontend-backend integration.
